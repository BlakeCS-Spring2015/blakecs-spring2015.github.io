# While Loops


  
This is a while loop.
  
<br>

<!-- arbitrary comment -->

    x = 0
    while x <= 10:
      print x
      x = 1 + x

<br>

If x is less then or equal to 10, the loop will print x, then add one to x, then repeat until x is equal to 10.
